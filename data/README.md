# Dataset Information

The dataset used in this project is not included in this repository due to its large size.

Dataset Details:
- 460,292 patient records
- 20 clinical and demographic features
- Target: Lung Cancer Prediction (Yes/No)

To run this project:

1. Download the dataset from the provided link below.
2. Place the file inside the `data/` folder.
3. Ensure the filename is:

   lung_cancer_prediction.csv

Dataset download link:
